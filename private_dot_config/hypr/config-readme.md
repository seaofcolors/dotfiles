# Introduction

This readme file explains a few easy things that are to be done besides editing the config files of hyprland, waybar, etc.

There are four applications that are necessary to create a unified look and feel. If KDE was installed previously, the KDE system settings will have to be edited, too.


# Prerequisites

Install
> kvantum kvantum-qt5 kvantum-theme-catppuccin-git
> qt6ct qt5ct catppuccin-qt5ct-git
> catppuccin-gtk-theme-mocha
> catppuccin-sddm-theme-mocha
> catppuccin-mocha-grub-theme-git
> nwg-look nwg-icon-picker
> papirus-icon-theme ttf-jetbrains-mono-nerd inter-font


# Setup

Open qt5ct, qt6ct, kvantum-manager and nwg-look and set the following:
- Theme: Catppuccin mocha lavender
- Regular font: Inter 12
- Monospace font: Jet Brains Mono [Nerd Font Mono] 12
- Icon theme: Papirus dark

If KDE system settings app is installed, also adjust the fonts and icons there.
- Small 8pt, Toolbar 10pt, Menu 10pt, Window title 10pt (all Inter)
